Need an App Password?

For security, Gmail requires an App Password rather than your primary password. Here is how to create one in 2 minutes:

  1. 1 Open Google Account Settings .
  2. 2 Search for or navigate to 2-Step Verification and enable it.
  3. 3 Search for App passwords in the search bar.
  4. 4 Enter a name (e.g. CareerRadar) and click Create.
  5. 5 Copy the generated 16-character code and paste it above.
